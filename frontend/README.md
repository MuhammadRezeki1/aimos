# AIMOS Frontend

**AIMOS** stands for **Artificial Intelligence Monitoring and Observation System**.

This is a clean white professional Next.js frontend starter for an AI-based monitoring, scraping, summarization, and analysis platform.

## Features

- Next.js App Router structure
- Hardcoded login page
- Top navbar with hover dropdowns
- AIMOS SVG logo and brand mark
- Dashboard home with Trend Monitor, Post Topics, and Creators
- Explore menu pages
- Entity Analysis menu pages
- Deep Analysis menu pages
- Clean white UI with soft cards and professional typography
